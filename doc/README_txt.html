    Running the Application:

    1. Please make sure that the Devise gem is installed and the following

    commands are executed:

    > rails g devise:install > rails g devise user

    1. Run migrations and load the seed file:

    > rake db:migrate > rake db:seed

    1. Start the Webrick server:

    > rails s

    1. Test the api by visiting a few endpoints:

    localhost:3000/api/v1/users/1/lists localhost:3000/api/v1/users/2/lists localhost:3000/api/v1/users/1/lists/2/items/2 localhost:3000/api/v1/users/3/lists/9/items/9

    etc.
