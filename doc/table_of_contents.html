    • Steps that are necessary to get the application up and running:
    • Ruby version: ruby 2.1.5p273
    • Rails version: Rails 4.2.1
    • Database creation and initialization: rake db:migrate && rake db:seed
    • Deployment instructions:
    • Please make sure that the Devise gem is installed and the following commands are executed:
    • > rails g devise:install
    • > rails g devise user
    • Run the bundle, migrations and load the seed file:
    • > bundle install
    • > rake db:migrate
    • > rake db:seed
    • Start the Webrick server:
    • > rails s
    • Test the api by visiting a few endpoints:
    • localhost:3000/api/v1/users/1/lists
    • localhost:3000/api/v1/users/2/lists
    • localhost:3000/api/v1/users/1/lists/2/items/2
    • localhost:3000/api/v1/users/3/lists/9/items/9
